AgentWeb Architecture

Web access CLI for AI agents — search, fetch, research, crawl, and source-pack

Key Design Decisions

  • • No API keys required for search/fetch — zero-config baseline
  • • Layered extraction: HTTP → Jina → Camoufox → Agent Browser
  • • 14 parallel search providers with subject-aware routing
  • • Zero external API dependencies — no API keys required
  • • Persistent auth profiles via Camoufox (file-locked, fingerprinted)

Data Flow Pipeline

  • • search → parallel providers → rank + balance → enriched JSON/MD
  • • fetch → layered extraction → text/markdown/structured data
  • • research → search → fetch top → quality score → answer pack
  • • crawl → BFS seed → content-area links → answer pack
  • • All commands zero LLM — keyword-based evidence extraction

Deployment & Dependencies

  • • Core: requests ≥ 2.33 (zero optional deps for basic usage)
  • • Browser: camoufox + playwright (optional [browser] extra)
  • • LLM: None — zero external API dependencies
  • • Extraction: trafilatura (optional, richer markdown output)
  • • Install: uv tool install or pip, works on macOS/Linux
